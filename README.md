## ▶️ Cómo ejecutar cualquier proyecto

**Requisitos:** tener Java instalado.
```bash
# Entrar a la carpeta del proyecto
cd 01-reloj

# Compilar todas las clases
javac *.java

# Ejecutar
java Principal
```

## 📚 Conceptos cubiertos
- Clases, objetos y constructores
- Encapsulación: atributos privados, getters y setters
- Herencia y sobreescritura de métodos (`@Override`)
- Sobrecarga de constructores y métodos
- `ArrayList` con objetos propios
- Validación de datos en setters
- Hilos básicos con `Thread.sleep()`
